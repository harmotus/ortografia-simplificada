REGLAS

Reglas : Versión 1 ( Base : Bibliografía )

• 1.  La letra 'v' se reemplaza por la letra 'b'.

• 2.  La letra 'c' se reemplaza por la letra 'z' en las combinaciones 'ce' y 'ci'.

• 3.  La letra fuerte 'g' se reemplaza por la letra 'j' en las combinaciones 'ge' y 'gi'.

• 4.  El dígrafo 'gu' se reemplaza por la letra suave 'g' en las combinaciones 'gue' y 'gui'.

• 5.  El dígrafo 'qu' se reemplaza por la letra 'k' en las combinaciones 'que' y 'qui'.

• 6.  La letra 'q' y la letra 'c' se reemplazan por la letra 'k'.

• 7.  La letra vocal 'y' se reemplaza por la letra vocal 'i'.

• 8.  La letra muda 'h' y la diéresis 'ü' desaparecen.

• 9.  La letra 'r' conserva sus reglas tradicionales.

• Subir  Base

Reglas : Versión 2 ( Base : Versión 1 )

• 1.  El dígrafo 'ch' se reemplaza por la letra 'c'.

• 2.  El dígrafo 'll' se reemplaza por la letra 'h'.

• 3.  El dígrafo 'rr' se reemplaza por la letra 'v'.

• 4.  La letra fuerte 'r' se reemplaza por la letra 'v'.

• 5.  La letra 'm' se reemplaza por la letra 'n' en las combinaciones 'mb', 'mp' y 'mn'.

• 6.  La letra 'x' se reemplaza por las letras 'k','s','j' o la combinación 'ks' según el contexto.

• Subir  Base

RESUMEN

• 1.  La letra 'k' siempre se pronuncia como la letra común 'k'. ( v1 ) { letra 'k' = fonema /k/ }

• 2.  La letra 'g' siempre se pronuncia como la letra suave 'g'. ( v1 ) { letra 'g' = fonema /g/ }

• 3.  La letra 'r' siempre se pronuncia como la letra suave 'r'. ( v2 ) { letra 'r' = fonema /ɾ/ }

• 4.  La letra 'v' siempre se pronuncia como el dígrafo 'rr'. ( v2 ) { letra 'v' = fonema /r/ }

• 5.  La letra 'h' siempre se pronuncia como el dígrafo 'll'. ( v2 ) { letra 'h' = fonema /ʎ/ }

• 6.  La letra 'c' siempre se pronuncia como el dígrafo 'ch'. ( v2 ) { letra 'c' = fonema /t͡ʃ/ }

DETALLE

• 1.  El fonema /b/ está bien representado por la letra 'b'.

• 2.  El fonema /k/ puede ser representado por la letra 'k' o la letra 'c'.

• 3.  En la versión 2 el dígrafo 'ch' puede ser representado por la letra 'c' o la letra 'k'.

• Subir  Base

FILOSOFÍA

• 1.  Se escribe como se pronuncia y se pronuncia como se escribe.

• 2.  Toda letra denota un solo fonema ( que es denotado por una sola letra )

• 3.  Las palabras pueden tener más de una sílaba tónica. Por ejemplo: tíorío, marímarí, etc.

• 4.  Las palabras pueden escribirse separadas en sílabas. Por ejemplo: sub.lu.nar, al.le.to, etc.

• 5.  Explorar la cantidad máxima de sílabas que se pueden pronunciar con los fonemas del idioma.

• 6.  Explorar la cantidad máxima de palabras que se pueden pronunciar con las sílabas del idioma.

• 7.  Poder usar una ortografía formal establecida por una academia internacional, regional o nacional.

• 8.  Poder usar una ortografía fonética débil basada sólo en fonemas y usando un alfabeto comunitario.

• 9.  Poder usar una ortografía fonética fuerte basada sólo en alófonos y usando un alfabeto comunitario.
